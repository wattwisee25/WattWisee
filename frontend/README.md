WattWisee

=================

This project was generated using Angular CLI version 20.0.5
(https://github.com/angular/angular-cli).

--------------------------------------------------
Development Server
--------------------------------------------------

To start a local development server:

cd backend
node server.js

cd frontend
ng serve

Open your browser at http://localhost:4200/.
The application will automatically reload whenever you modify any source files.

--------------------------------------------------
Code Scaffolding
--------------------------------------------------

To generate a new component:

ng generate component component-name

For a complete list of available schematics
(components, directives, pipes, etc.):

ng generate --help

--------------------------------------------------
Building the Project
--------------------------------------------------

To build the project:

cd frontend
ng build

The build artifacts will be stored in the dist/ directory.
The production build is optimized for performance and speed.

--------------------------------------------------
Testing
--------------------------------------------------

Unit Tests:
cd frontend
ng test

End-to-End (E2E) Tests:
cd frontend
ng e2e

> Angular CLI does not include an E2E testing framework by default.
  Choose the one that best suits your needs.

--------------------------------------------------
Backend
--------------------------------------------------

To start the Node.js backend:

cd backend
node server.js
